# Quick-access-bar-

Quick Access Bar Expansion for NeoForge on Minecraft 1.21.1.

## Build

Use JDK 21 for this project.

If you already have Gradle available, run:

```powershell
gradle build
```

If you generate the wrapper first, you can use:

```powershell
.\gradlew.bat build
```

## Current status

This repository is still an early prototype. The UI and hotbar switching flow are implemented, but persistence and multiplayer-safe syncing still need more work.
